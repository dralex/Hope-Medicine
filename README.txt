
Программа-симулятор для моделирования медицины на игре "Надежда" от
мастерской группы Лестница в небо. Программа позволяет проводить анализ
пациента -- цитологию, гормональный фон и генетические отклонения.
Клиентская программа запускается на компьютере врачей, а серверная часть --
у мастеров. Анализы берутся из кода, получаемого от исследуемого человека.
Одного кода ("пробирки с кровью") хватает только на ограниченное число
анализов, именно для учета этого и требуется серверная часть, на которой
ведется учет.

Игровые материалы по деталям медицинских моделей можно найти на сайте http://fedoseev.net/rpg/hope/

Клиентская часть программы требует библиотеку Shooes (http://shoesrb.com/) для запуска.
Северная часть программы -- простое CGI-приложение.
